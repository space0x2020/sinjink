(1) A + B × C

　　　　B × C → B C ×

(2) A × (B + C)

　　　　(B + C) → B C +

A × B C + → A B C + ×

(3) (A+B) ÷ (C -D)

(A + B) → A B +

　　　　(C - D) → C D -

A B + ÷ C D - → A B + C D - ÷

(4) X = A + B － C

　　A + B → A B +

A B + - C → A B + C -

[3] スタック

スタックはコンピュータで用いられる基本的なデータ構造の1つで、データを後入れ先出し（LIFO: Last In First Out; FILO: First In Last Out）の構造で保持するものである。(ウィキペディア)

データを格納する　→ PUSH

データを取り出す　→ POP
