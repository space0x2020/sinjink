SQL問題(2)

社員B（社員番号, 部門名, 社員名, 役割名）

ア　SELECT 部門名 FROM 社員B
GROUP BY 部門名
HAVING COUNT(*) >= 3
WHERE 役割名 = 'プログラマ'

イ　SELECT 部門名 FROM 社員B

GROUP BY 部門名

ウ　SELECT 部門名 FROM 社員B

SQL問題(5)

“商品”表、”在庫”表に対する次のSQL文の結果と同じ結果が得られるSQL文はどれか。 ここで、下線部は主キーを表す。 
SELECT  商品番号 FROM 商品
     WHERE 商品番号 NOT IN (SELECT 商品番号 FROM 在庫)

商品
商品番号 商品名 単価

在庫
倉庫番号 商品番号 在庫数

ア　SELECT 商品番号 FROM 在庫
WHERE EXISTS (SELECT 商品番号 FROM 商品)

イ　SELECT 商品番号 FROM 在庫
WHERE NOT EXISTS (SELECT 商品番号 FROM 商品)

ウ SELECT 商品番号 FROM 商品
WHERE EXISTS (SELECT 商品番号 FROM 在庫
WHERE 商品.商品番号 = 在庫.商品番号)

エ　SELECT 商品番号 FROM 商品
WHERE NOT EXISTS (SELECT 商品番号 FROM 在庫
WHERE 商品.商品番号 = 在庫.商品番号）

副問合せ(SELECT 商品番号 FROM 在庫)は、在庫にある商品番号を抽出している。
主問い合わせは、NOT IN (副問合せ)であるので、在庫に無い商品を抽出している。

ア　SELECT 商品番号 FROM 在庫
WHERE EXISTS (SELECT 商品番号 FROM 商品)

副問合せが(SELECT 商品番号 FROM 商品)なので、商品表に行があれば、
EXISTS句は常にTRUEとなり、在庫表にあるすべての商品番号が抽出される。
→×

イ　SELECT 商品番号 FROM 在庫
WHERE NOT EXISTS (SELECT 商品番号 FROM 商品)

副問合せが(SELECT 商品番号 FROM 商品)なので、商品表に行があれば、
NOT EXISTS句は常にFALSEとなり、在庫表から何も抽出されない。
→×

ウ SELECT 商品番号 FROM 商品
WHERE EXISTS (SELECT 商品番号 FROM 在庫
WHERE 商品.商品番号 = 在庫.商品番号)

相関副問合せにより、商品表にある商品番号と同じ商品番号が在庫表に存在すれば EXISTS句はTRUEとなる。
商品表のうち、在庫表に存在する商品番号を抽出している。
→×

エ　SELECT 商品番号 FROM 商品
WHERE NOT EXISTS (SELECT 商品番号 FROM 在庫
WHERE 商品.商品番号 = 在庫.商品番号）

相関副問合せにより、商品表にある商品番号と同じ商品番号が在庫表に存在しなければば NOT EXISTS句はTRUEとなる。
商品表のうち、在庫表に存在しない商品番号を抽出している。
→正解。
