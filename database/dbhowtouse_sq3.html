sqlite3の起動方法

  1. 「スタート」右クリック→「Windows PowerShell」をクリックします。Windows PowerShell ウィンドウが表示されます

  2. Windows PowerShell ウィンドウが表示されたら、sqlite3 フォルダーへ移動します。

    PS C:\Users\PANDA-ID> cd Desktop\sqlite3 <ENTER>    ←
PS C:\Users\PANDA-ID\Desktop\sqlite3> _

  3. sqlite3 を起動します。

    PS C:\Users\PANDA-ID\Desktop\sqlite3> .\sqlite3.exe .\mydb.db <ENTER>    ←
-- Loading resources from C:\Users\PANDA-ID/.sqliterc
SQLite version 3.35.4 2021-04-02 15:20:16
Enter ".help" for usage hints.
sqlite> _

sqlite3の終了方法

  1. sqlite3のプロンプト(sqlite>)上で、.q を入力します。 
    sqlite> .q <ENTER>    ←
PS C:\Users\PANDA-ID\Desktop\sqlite3> _
  2. PowerShellを終了します。 
    PS C:\Users\PANDA-ID\Desktop\sqlite3> exit <ENTER>    ←

SQL文の実行

  • sqlite3では、SQL文の後ろにセミコロン(;)を付けます。(以降、<ENTER>は省略します。) 
    sqlite> select * from dept;    ←最後に ; を付ける。
deptno  dname       loc
------  ----------  ----------
   10   ACCOUNTING  NEW YORK
   20   RESEARCH    DALLAS
   30   SALES       CHICAGO
   40   OPERATIONS  BOSTON
sqlite> _

テーブル定義の参照

  • 表の定義を参照するには、「.schema 表名」コマンドを使います。 
    sqlite> .schema dept
CREATE TABLE dept (
    deptno  int constraint pk_dept primary key,
    dname   varchar(14),
    loc     varchar(13)
);
sqlite> _

長いSQL文の入力

  • セミコロン(;)を付けずにSQL文の途中で、<ENTER> を入力すると、プロンプトが「...>」になり継続して入力できます。 セミコロン(;)を入力して<ENTER>を入力すると、SQL文が実行されます。 
    sqlite> select * <ENTER>
   ...> * <ENTER>
   ...> from <ENTER>
   ...> dept <ENTER>
   ...> ; <ENTER>
deptno  dname       loc
------  ----------  ----------
   10   ACCOUNTING  NEW YORK
   20   RESEARCH    DALLAS
   30   SALES       CHICAGO
   40   OPERATIONS  BOSTON
sqlite> _
