演習４－１

[1] 社員表から

→ from 社員表

→ from 社員表 where 部署コード = '0003'

[3] 行を抽出 (列の指定がないので全列とする)

→ select * from 社員表 where 部署コード = '0003'

演習４－３

社員表と部署表を部署コードで結合し、社員番号、社員名、部署名を抽出しなさい。列見出しは、番号、氏名、所属とすること。

[1] 社員表と部署表を結合。社員表を s、部署表を b と別名をつける。

→ from 社員表 s join 部署表 b

[2] 社員表と部署表を部署コードで結合

→ from 社員表 s join 部署表 b on s.部署コード = b.部署コード

[3] 社員番号、社員名、部署名を抽出

→ select s.社員番号,  s.社員名,　b.部署名 from …

[4] 列見出しは、番号、氏名、所属とする

→ select s.社員番号 as "番号", s.社員名 as "氏名", b.部署名 as "所属"
from 社員表 s join 部署表 b on s.部署コード = b.部署コード

実行例：
> select s.社員番号 as "番号", s.社員名 as "氏名" , b.部署名 as "所属"
  from 社員表 s join 部署表 b on s.部署コード = b.部署コード;

演習４－４

社員表と部署表から、部署名が「総務部」である人の社員番号、社員名、部署コードを抽出しなさい。

[1] 社員表と部署表から→社員表と部署表を結合(社員表を s、部署表を b として)

→ from 社員表 s join 部署表 b on s.部署コード = b.部署コード

[2] 部署名が「総務部」→抽出条件として 部署表．部署名 = '総務部'

from 社員表 s join 部署表 b on s.部署コード = b.部署コード
→ where b.部署名 = '総務部'

[3] 社員番号、社員名、部署コードを抽出

→ select s.社員番号, s.社員名, s.部署コード
from 社員表 s join 部署表 b on s.部署コード = b.部署コード
where b.部署名 = '総務部'

実行例：
> select s.社員番号, s.社員名, s.部署コード
  from 社員表 s join 部署表 b on s.部署コード = b.部署コード
  where b.部署名 = '総務部';

------    ----      -----

000005    高橋五郎  0001
