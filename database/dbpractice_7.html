演習問題解答例：

演習７－１

次に定義される”SHOUHIN”表を作成しなさい。

列名 型 制約
商品コード CHAR(3) 主キー
商品名 VARCHAR(24)
単価 NUMERIC(8) 

実行例：
> create table SHOUHIN (
  商品コード char(3) primary key,
  商品名     varchar(24),
  単価       numeric(8)
  );

演習７－２

“SHOUHIN”表に以下のデータを挿入しなさい。

商品コード 商品名 単価
S01 鉛筆 30
T03 テレビ 40,000
S02 お茶 230
 

実行例：
> insert into SHOUHIN(商品コード, 商品名, 単価) values('S01', '鉛筆', 30);
> insert into SHOUHIN(商品コード, 商品名, 単価) values('T03', 'テレビ', 40000);
> insert into SHOUHIN(商品コード, 商品名, 単価) values('S02', 'お茶', 230);

確認：
> SELECT * FROM shouhin;


商品コード  商品名  単価
-----       ---     -----
S01         鉛筆    30
T03         テレビ  40000
S02         お茶    230

演習７－３

“SHOUHIN”表の商品名と単価のみを抽出するビュー V_SHOUHINを定義しなさい。

[1] SHOUHIN表の商品名と単価のみを抽出するSELECT文は、

→ SELECT 商品名, 単価 FROM SHOUHIN

[2] CREATE VIEW .. AS SELECT ...

→ create view V_SHOUHIN as SELECT 商品名, 単価 FROM shouhin

実行例：
> CREATE VIEW v_shouhin AS SELECT 商品名, 単価 FROM shouhin;

演習７－４

V_SHOUHINビューから、単価が1000円以下のデータを抽出しなさい。

[1] V_SHOUHINから

→ from V_SHOUHIN

[2] 単価が1000円以下　選択条件：単価列の値が1000以下

→ from V_SHOUHIN where 単価 <= 1000

[3] 行を抽出 (列の指定がないので全列とする)

→ SELECT * FROM v_shouhin WHERE 単価 <= 1000

実行例：
> SELECT * FROM v_shouhin WHERE 単価 <= 1000;


商品名  単価
---     ---
鉛筆    30
お茶    230
