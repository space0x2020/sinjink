[1] DDL文例 CREATE VIEW文

■ ビュー：射影、選択、結合などの一部を抽出したり、複数の表に対して検索や表の結合を行った際の結果データが取り込まれる新たな表。

ビューの作成
create view ビュー名 as select ステートメント

ビューの削除
drop view ビュー名 

実行例１

> create view 社員表V
  as select a.社員名, b.部署名 from 社員表 a join 部署表 b
  on a.部署コード = b.部署コード


実行例２

> select * from 社員表V
社員名   部署名
----     ---
山田一郎 営業部
鈴木花子 総務部
佐藤三郎 開発部
田中良子 開発部
高橋五郎 総務部


実行例３

> drop view 社員表V
